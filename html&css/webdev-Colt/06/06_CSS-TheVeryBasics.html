What is CSS

CCS allows us to control the style of an HTML page, but first we need the HTML page. CSS stands for cascading style sheets and is not a full on programming language, it is a language though to describe the visual aspects of our HTML document. Whenever we write CSS, we writing what is known as CSS rules. Almost everything you do in CSS follows this basic pattern.

Generic example

  • selector {
    property: value;
    }

More real example

  • h1 {
    color: purple;
    }

Looking at the examples you see that you have something known as selector, for example h1, a property being color and a value for the said property. So in the more real example you see all h1 headings, which can only be 1, has the text color of purple. However it can become more complicated, you can add more properties or a element selector with a attribute selector as well furthuring that.

Including styles correctly

There are a couple ways to include css styles. You can write styles directly inline with an element, however this is never a good idea. An example of that would be <h1 style="color:purple">. Another way is to use the style element which can be placed in the head of the HTML document. The last way and highly recommended, is creating a .css file and then link it in the HTML document using the link element. An example of a link element would look like this <link rel="stylesheet" href="app.css">, where app.css would be the app.css file you use for styling your document.

Color & Background: Color Properties

The Color property changes the color of the text while background-color property, which as it says will change the background color of any given selector, note you also get a property background which can change the background color but can do many other things as well.

Color systems: RGB, Named & Hexadecimal colors

The color property in css has a range of different color units it accepts, such as a name like pink or orange, it can also take in hexadecimal values as so #ffffff, note that with hexadecimal if the 3 pairs are the same value, you can write it in a shorter manner like this #fff or for example #cc55ee can be written like #c5e. CSS also accepts rgb input like so rgb(214,122,127) note values need to be in parenthesis as well as hsl colors hsl(30, 100%, 50% , .3). The color property will change the text color and not the backgroung.

Common text properties

There are various ways you can style text in CSS, this can range from aligning text to the center of the page as so, text-align: center.Earlier when discussing HTML we noted that its bad practice to use elements like <b> and to rather use CSS, and here is an example of bolding with the font weight porperty font-weight: 100 - 900 or another example is to use words for the font weight like so font-weight: thin, normal,bold or bolder. You can go as far as placing an underline under the text using the text decoration property text-decoration: plum wavy underline or even just adjust the line height or letter spacing like so line-height: 1.2em, letter-spacing: 5px.

The font family property

This is an example of the font family property font-family: Ariel, Helvetica, Sans-serif it is important to note the not all browsers share the same fonts, this is also true across operating systems like Windows, Mac and linux. This is why we would set a font family, so in case they dont have one specific font, it will fallback to the next available font or font family.
