Intro to Functions

  • let die1 = Math.round(Math.random() * 6)
    let die2 = Math.round(Math.random() * 12)
    let die3 = Math.round(Math.random() * 20)
    let die4 = Math.round(Math.random() * 6)

  • let die1 = rollDice(6)
    let die2 = rollDice(12)
    let die3 = rollDice(20)
    let die4 = rollDice(6)

What is a function?. A function is re-useable code which is wrapped up and given a name. Not all functions have names as we will see later on. The code only needs to be written once.

Our Very First Function

  • Defining a Function
    • function funcName (parenthesis) {
      return your code here;
      }

  • Running a Function:
    • funcName();

The syntax for a function is shown above, both for defining and running. Its important note that when it comes to functions that there are 2 steps involved. First, is defining the function and Second, is running the function.

Defining a function involved you telling JavaScript what is a function and providing the code it will run. While running the function only requires you to state the function name.

Functions can also accept a type of input which affects the outcome or result of the code. This is seen later on, known as arguments whereby you enter parameters inside the parenthesis.

Arguments Intro

  • function funcName (firstName) {
    console.log(`Hi, ${firstName}`)
    }

Arguments is a fancy way of saying input into a function. Seeing the example above, inside the parenthesis is where you define your variable name or what we call parameter.

Arguments is nothing new to us. We have used arguments before in methods like .indexOf('h') or arr.push('element').

Methods and Functions are the same thing.

Something to note is, when a function is expecting a parameter and its blank, it will output the result as undefined.

Functions With Multiple Arguments

  • function multiply (firstName, lastName) {
    console.log(`Hi, ${firstName} ${lastName[0]}`).;
    }

You can define functions that accepts multiple aruguments. You separate them by a , comma. Its also important to note that order does matter when it comes to multiple arguments.

The Return Keyword

  • function multiply (num1, num2) {
    return num1 * num2;
    }

With the return keyword, there are two important things to remember.

First, is that return allows us to save the output of the function to a variable. Up until now we have seen examples using console.log(), but the example above allows us to capture the result.

Second, is that when the code reaches the return keyword it will stop the code from Running.
