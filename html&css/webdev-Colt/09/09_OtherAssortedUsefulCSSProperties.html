The Position Property

  • position: static/relative/absolute/fixed/sticky;
    top: 50px;
    right: 10%;
    bottom: 4em;
    left: 0;

Above shows the property position and other properties that control the possitioning once a element has a position value set.

Static

  • position: Static;

An element with a value of static, will remain positioned to the normal flow of the document. The top, right, bottom and left properties will have no affect on this value, as this is the default value.

Relative

  • position: relative;

An element set with a relative position value, is positioned according to the normal flow of the document, however offset by the top, right, bottom and left properties relative to itself.

Absolute

  • position: absolute;

The element with the value of absolute is removed from the normal document flow, and no space is created for the element in the page layout. It is positioned to its closest already positioned relative. Otherwise it is relative to it's containing block, in other words the <body> element of the page. The final outcome or position is determined by the top, right, bottom and left properties.

Fixed

  • position: fixed;

An element with the position fixed, works similair to that of absolute. It is removed from the normal flow of the document, however is only relative to the containing block or <body> element. The one difference is the that it will remain in said position given by the top, right, bottom and left properties even when scrolling down and it will remain fixed to an element that also has a scrolling mechanism.

Sticky

  • position: sticky

A sticky element unlike fixed remains in the flow of the document and is offset in whatever postion it was given by the top, right, bottom and left properties. However sticky will stick to its nearest ancestor that has a scrolling mechanism.

CSS Transitions

  • transition: property/duration/transition-timing-function/delay;
    transition: color 0.25s ease-in 0.5s, border-color 0.25s ease-in 0.5s;
    • transition-property: color;
    • transition-timing-function: ease-in;
    • transition-delay: 0.5s;
    • transition-duration: 0.25s;

Transitions allow us to transition one property value to another property value. The transition property is a shorthand just like the border shorthand property.
