Introduction to HTML

Taking a look at this saying, the Purple Dino Danced. We can break down what HTML is compared to and how to explain what HTML is. HTML is the 'what'in this sentence, in otherwords the dino, whereas the purple is css and danced is js. HTML is what is being displayed or viewed, so the in short its the what.

HTML is known as a markup language and stands for Hypertext Markup Language. It is not a full blown programming language with complex logic, It's a way to mark up a document (plain text) and have a way to indicate what is bold, italic or what is a link as an example, through what is known as elements or HTML elements.

To write HTML we use what is know as HTML elements, as exmaple:

  • <p>
    • This represents the paragraph element
  • <h1>
    • This represents the main header on the page
  • <img>
    • This embeds an image
  • <form>
    • This represents a form
These are some of the elements used to write HTML. Although above doesn't illustrate this, but we generally wrap some form of content with what is knowns as <opening tag> and a </closing tag>.

An example would be something like this <h1>This is a header</h1> to represent the start and end of the main header. You can also nest tags within elements, using the same example as before, you can make the header bold, like so <h1><b>This is a header in bold</b></h1>.

Paragraph Elements

The first element we look at is the <p> for paragraph, and this doesn't mean its only for text purposes, you can have a paragraph for things you wish to group together. That could be a paragraph of images or different pieces of a form. However for now we are only focused on textual paragraphs.
